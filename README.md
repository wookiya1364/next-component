## Getting Started

서버를 켜야 합니다.

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

[http://localhost:3000](http://localhost:3000) 접속해서 확인하세요.


## Commit Messages Style

|타입|설명|
|-----|-----|
|feat|새로운 기능 추가|
|fix|버그 수정|
|docs|문서 수정|
|refactor|코드 리펙토링|
|test|테스트 코드 추가|
|perf|성능 개선|
|design|디자인 변경|
